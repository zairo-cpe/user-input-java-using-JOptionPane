# JavaUserInput-JOptionPane
This is a Java input and output with age and name using JOptionPane, making it a GUI program.

# Details:
This program is a Java program that takes input from the user which are name and age.
The program is GUI-based, which means it does not run in the console, but does add output as updates for every action made in the program.
The program used JOptionPane, which is a native library of Java. To import the library, I used "javax.swing.JOptionPane".
This program serves as my progress in learning Java as one of my subjects here at college.
